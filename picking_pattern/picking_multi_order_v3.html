SPEED

Multi Order Picking とは

複数オーダーを同時並行で処理する方式。
作業者が SKU Bin を1回訪問するだけで、
その Bin に必要な全オーダー分をまとめてピックし、
それぞれの Order Box へ即時振り分けます。

効率評価（①との比較）

Bin 訪問回数：4回（① Single は12回）
ピック総数：12回（同じ）
同時処理：あり（最大4オーダー並列）

✓ 長所 移動距離が大幅に短縮、スループット向上
× 短所 作業が複雑、誤ピックリスク増
