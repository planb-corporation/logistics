/ Inventory Import とは

上位 WMS のハンディを使って Bin に積み付け。
「どのBinのどのスロットに何がいくつ」という
データを CSV / API で ASRS に連携します。

/ 特徴

積み付け管理：上位 WMS 側（RR ハンディ不要）
データ連携：CSV アップロード or API
想定用途：上位 HT 使用・大量ロケ移動

✓ 上位 WMS 主体で完結できる
× CSV / API 連携の仕組みが必要
