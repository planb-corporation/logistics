Stowing Patterns

ASRSへの商品格納における4つの入庫方式のシミュレーションです。

使い方 各カードをクリック → アニメーション画面で ▶ Start を押してください
/ 4 Patterns 4つの格納方式
Pattern 01
Berth Receiving
berth receiving
入荷検品

ASN（入荷予定）と商品を照合しながらBinに積み付け。管理者確定後にASRSへ格納。

ASN照合あり
管理者承認あり
作業場所フロア
アニメーションを見る→
Pattern 02
Item Receive
item receive
在庫登録

ASN照合なしで商品をBinに積み付け。確定操作後に即座にASRSへ格納可能。

ASN照合なし
管理者承認なし
作業場所フロア
アニメーションを見る→
Pattern 03
Inventory Import
inventory import
外部データ連携による在庫登録

上位WMSで積み付けデータを作成しCSVでASRSに連携。Binをステーションに持ち込み格納。

ASN照合上位WMS側
データ連携CSV / API
作業場所フロア
アニメーションを見る→
Pattern 04
Item Stow
item stow
ステーションでの商品格納

空BinをASRSから自動供給。ステーション上で商品を積み付けそのまま格納。床作業不要。

Bin取り出し自動供給
管理者承認なし
作業場所ステーション
アニメーションを見る→
