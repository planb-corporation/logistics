/ Pattern 04 Item Stow — ステーションでの商品格納
④ Item Stow

空 Bin を ASRS から自動供給 → ステーション上で商品を積み付け → そのまま ASRS へ格納（フロア作業不要）

Speed

/ Item Stow とは

空 Bin を ASRS から自動でステーションに供給。
作業者はステーション上で商品を Bin に積み付け、
完了後はそのまま AMR が Bin を回収して ASRS に格納します。

/ 特徴（①②③ との比較）

作業場所：ステーション（フロア作業なし）
Bin 取り出し：なし（ASRS から自動供給）
想定用途：少量急ぎ・重量物 Bin

✓ 床作業ゼロ・重量物の取り回し不要
× 同時作業人数はステーション数が上限
