/ Pattern 01 Single Order Picking
① Single Order

1オーダーずつ順番に処理 / 作業者が SKU Bin を巡回し、各 Order Box へ 1点ずつ投入

Speed

/ Single Order Picking とは

1オーダーずつ順番に処理する方式。
作業者が SKU Bin からアイテムを取り出し、
対応する Order Box へ1点ずつ投入します。
Order 1 → 2 → 3 → 4 の順に直列処理。

/ 効率評価

ピック回数：12回（4オーダー × 各3アイテム）
Bin 訪問回数：12回（各 Bin を複数回訪問）
同時処理：なし（直列処理）

✓ 長所 誤ピックが少なく管理がシンプル
× 短所 移動距離が長く、スループット低め
