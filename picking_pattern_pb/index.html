Picking Patterns

ピッキングステーションにおける3つの作業方式のシミュレーションです。

使い方 各カードをクリック → アニメーション画面で ▶ START を押してください
/ 3 Patterns 3つのピッキング方式
Pattern 01
Single Order
シングルオーダーピッキング

1オーダーずつ順番に処理する基本方式。作業者が SKU Bin を巡回し、1アイテムずつ対応する Order Box へ投入。シンプルで誤ピックが少ない。

Bin 訪問回数 12 回
同時処理 なし（直列）
スループット 低
アニメーションを見る →
Pattern 02
Multi Order
マルチオーダーピッキング

複数オーダーを同時並行で処理する方式。1回の Bin 訪問で複数オーダー分をまとめてピックし、即時振り分け。移動距離を大幅に削減。

Bin 訪問回数 4 回
同時処理 あり（最大4オーダー）
スループット 中〜高
アニメーションを見る →
Pattern 03
Total Pick
トータルピック（種蒔き方式）

全アイテムを一括収集後、Sorter が自動仕分けする2フェーズ方式。Phase 1 で DAS/GAS へ全品目を集約し、Phase 2 で自動振り分け。大量オーダーに最適。

Bin 訪問回数 4 回
仕分け Sorter 自動処理
スループット 高
アニメーションを見る →
